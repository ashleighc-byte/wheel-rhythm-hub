## How to Use These Files in Airtable

1. Create a base called **"Freewheeler HQ"**
2. For each file, create the corresponding table listed in the "Airtable Table Target" column above
3. Import each file's structured tables as records
4. Link tables using the relationship fields noted in each file
5. Set up views per table: Grid (data entry), Gallery (visual), Calendar (dates), Kanban (status tracking)
