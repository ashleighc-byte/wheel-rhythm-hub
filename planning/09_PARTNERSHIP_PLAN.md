## Partnership Strategy

Free Wheeler's programme delivery depends on a small set of critical partnerships. Our approach to partnerships is:
1. **Mutual value first** — every partner must be able to articulate what they get from the relationship
2. **Document everything** — all partnerships have written agreements, even if informal
3. **Grow with success** — pilot partnerships are intentionally lightweight; scale partnerships are more formal

| Field | Detail |

| Partner | VirtuPro Ltd (UAE) — operator of MyWhoosh virtual cycling platform |
| Type | Platform partner |
| Current Status | Active — students use MyWhoosh for virtual racing |
| Agreement Type | Informal / standard user licence |
| Value to Free Wheeler | Immersive virtual racing environment; free to use (currently); global platform |
| Value to MyWhoosh | User acquisition in school market; NZ brand building; case study for education sector |

**Partnership Development Roadmap:**

| Stage | Action | Timing |
|-------|--------|--------|
| Pilot (2026) | Establish formal relationship with MyWhoosh Partnerships team | May 2026 |
| Phase 1 (2026) | Request reduced-cost or free school licence agreement | July 2026 |
| Phase 2 (2027) | Explore API integration for automatic session sync | Q2 2027 |
| Phase 3 (2028) | Co-branded "Free Wheeler x MyWhoosh" league within platform | Q1 2028 |

**API Integration Goal (Phase 2):**
Currently, students must manually log sessions via the Free Wheeler app. The ideal state is:
- Student completes a MyWhoosh ride
- MyWhoosh pushes session data (duration, distance, elevation, power) to Free Wheeler via API
- Session auto-populates in the Free Wheeler app — no manual entry required
- This removes friction and increases session completion rates

### Sponsorship Proposal Process
1. Research sponsor's current community investment portfolio
2. Personalise the pitch deck (use data from pilot)
3. Send sponsorship proposal via email with deck attached
4. Follow up with a phone call within 5 business days
5. Offer a 30-minute presentation if interested
6. Negotiate terms and prepare sponsorship agreement
7. Co-create activation plan (what the sponsor does beyond writing a cheque)
